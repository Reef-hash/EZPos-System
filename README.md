# EZPos - Retail POS System

Sistem POS Windows Desktop untuk kedai runcit kecil. Dibina dengan C# .NET WinForms & SQLite.

## Ciri-ciri Utama
- Mod Jualan (Sales Mode)
- Mod Stok (Stock Mode)
- Imbasan barcode
- Cart & sistem pembayaran
- Pengurusan stok
- Cetakan resit ESC/POS (asas)

## Struktur Projek
- DataAccess: Logik database
- Models: Model data
- BusinessLogic: Logik bisnes
- UI: Antara muka pengguna (WinForms)

## Cara Guna (Ringkasan)
1. Buka projek dalam Visual Studio/VS Code
2. Pastikan NuGet package System.Data.SQLite telah dipasang
3. Build & run projek
4. Ikut panduan penuh dalam GUIDE-MY.md
